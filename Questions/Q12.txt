Что выведет данная программа и почему?

func main() {
    n := 0
    if true {
        n := 1
        n++
    }
    fmt.Println(n)
}

//

Программа выведет 0. В блок if программа зайдёт, но внутри этого блока
своя область видимости. Объявленная в этой области видимости переменная "n"
и действия над ней не будут видны за пределами области видимости блока if.
В функцию fmt.Println(n) будет передана переменная "n" из области видимости main.
