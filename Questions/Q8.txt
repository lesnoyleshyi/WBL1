В чем разница make и new?

//

new() выделяет динамическую память, забивает эту память нулями, возвращает
указатель на неё. С помощью new() можно выделить память для любого типа.

make() предназначена для создания "сложных" типов - map, slice, chan (и кастомных,
для которых эти типы являются базовыми). make() возвращает значение, а не указатель.
Необходимость в make() связна с тем, что map, slice и chan представляют собой
структуры, одним из полей которых является указатель на участок памяти.
Инициализация в го любого типа осуществляется его нулевым значением.
Для указателей нулевым значением является nil.
Попытка обращения к области памяти по nil-указателю приведёт к возникновению ошибки.
Пользователь лишён возможности напрямую обращаться к полю упомянутых типов,
содержащем указатель на область памяти. Следовательно, у него нет возможности
указать область памяти для использования объектом, память под который была
аллоцирована при помощи new().
Функция make() освобождает от необходимости таких сложных манипуляций.
Она выделяет память не только под структуру, представляющую "сложный тип",
но и под массив для этого типа.
