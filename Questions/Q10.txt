Что выведет данная программа и почему?

func update(p *int) {
    b := 2
    p = &b
}

func main() {
    var (
        a = 1
        p = &a
        )
    fmt.Println(*p)
    update(p)
    fmt.Println(*p)
}

//

1, потому что внутри функции мы изменили локальное значение указателя, однако,
внутри main указатель сохранил то же значение, ссылается в то же место
