Есть ли в Go перегрузка методов или операторов?

//

Нет, но есть "затенение" (shadowing):

Если тип "А" имеет метод hello(),
тип "В" имеет метод hello() и представляет собой структуру, встраивающую "A":
type B struct {
    A
},
то при вызове метода hello() от эекземпляра типа "В"
будет вызван метод типа "В", а не "А".

Тот же эффект будет в случае, если "наследование" имитировать не встраиванием,
а копированием типа:
type A int
func(a A) hello() { fmt.Println("hello, it's A") }

type B = type A //символ = позволяет копировать не только тип, но и методы
func(a B) hello() { fmt.Println("hello, it's B") }

При вызове метода hello() от эекземпляра типа "В"
будет вызван метод типа "В", а не "А".
